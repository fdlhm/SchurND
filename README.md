MATLAB codes accompanying:

Carlota M. Cuesta, Francisco de la Hoz, A non-recursive Schur-Decomposition Algorithm for $N$-Dimensional Matrix Equations, arXiv:2412.15840, (2024).

If you use this software in your research, please cite the above reference.

The program advdiftest.m requires the MATLAB functions herdif, herroots, and poldif, which are part of the MATLAB differentiation suite developed by J. A. C. Weideman and S. C. Reddy. These functions are available at:

https://appliedmaths.sun.ac.za/~weideman/research/differ.html
